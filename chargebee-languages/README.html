General instructions

  • You can enter/modify the translation for each field/attribute (called as ‘keys’ in the csv files).
  • Please DO NOT modify the field labels/ keys in any way.
  • Please DO NOT change the folder/file names as it WILL throw errors while uploading the pack.

Mandatory Folder

You need to provide translation ‘values’ in each file mandatorily. If the translation value is not entered, Chargebee will NOT be translating the field, it will appear in the default language.

Note: hosted_pages_themes.csv file contains themes related fields such as fonts etc, please do not translate these fields, however you can enter any specific font values if required.

Other fields are for reference only, you need not edit the details

Optional Folder

This folder contains the list of attributes which need not necessarily require translation as Chargebee provides the default translated values.

The default translations are present in the ‘value’ column.

However, you can overwrite/update the translation text if you want to.

This folder contains list of attributes of Hosted Pages and Customer Portal.

  • If you have updated the field values, then the fields will be listed in the csv files and you can provide a translated value for those fields, for instance, you have edited “checkout.page.title” from “Checkout” to “CART Checkout”
    • Chargebee will have default translation for the term “Checkout”
    • You should provide the updated translation for “CART Checkout”
  • If you have not updated the field translation, then the default translation will be shown (even if the field is modified in the application)
